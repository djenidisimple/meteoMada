# Flux utilisateur principaux

## Flux 1 : Consulter la météo du jour
1. Ouvrir l'app
2. Le GPS détecte la position
3. Afficher météo actuelle (cache ou API)
4. Voir les prévisions 7 jours
5. Option : voir météo marine si côtier

## Flux 2 : Rechercher une ville
1. Appuyer sur 🔍
2. Taper le nom ou utiliser le GPS
3. Sélectionner la ville
4. Voir les détails
5. Option : Ajouter aux favoris ⭐

## Flux 3 : Gérer les favoris
1. Appuyer sur ⭐
2. Voir la liste des favoris
3. Option : Supprimer un favori (swipe)
4. Option : Réorganiser (drag & drop)

## Flux 4 : Consulter une alerte cyclone
1. Recevoir une notification push
2. OU appuyer sur ⚠️
3. Voir la liste des alertes
4. Appuyer sur une alerte
5. Voir les détails : niveau, régions, consignes

## Flux 5 : Changer la langue
1. Appuyer sur ⚙️
2. Sélectionner Français ou Malagasy
3. L'interface se met à jour immédiatement
