# Charte graphique MeteoMada

## Inspiration
- Application météo native (iOS Météo, Android Weather)
- Palette tropicale inspirée de Madagascar
- Design simple, lisible, accessible

## Couleurs

| Nom | Hex | Usage |
|-----|-----|-------|
| Bleu océan | `#1B4F72` | Primaire, navigation |
| Orange soleil | `#F39C12` | Accents, alertes |
| Rouge cyclone | `#E74C3C` | Alertes critiques |
| Vert nature | `#27AE60` | Agricole, favorable |
| Jaune alerte | `#F1C40F` | Alertes modérées |
| Gris nuage | `#95A5A6` | Texte secondaire |
| Blanc cassé | `#F5F5F5` | Fond écran |
| Blanc pur | `#FFFFFF` | Cartes |

## Typographie
- **Titres** : Poppins Bold (20-28px)
- **Corps** : Inter Regular (14-16px)
- **Données** : Inter Medium pour les chiffres
- **Icônes** : Phosphor Icons (gratuites, cohérentes)

## Iconographie météo
Utiliser une police d'icônes météo (ex: Weather Icons) ou des SVG personnalisés :
☀️ Ensoleillé / ⛅ Partiellement nuageux / ☁️ Nuageux
🌧️ Pluvieux / ⛈️ Orageux / 🌊 Mer agitée
🌀 Cyclone / 🌾 Agricole
